# The-AI-Quiz
A Python application to test users on verifying if they are human or AI. It gives questions that humans can solve but AI can't answer. helps with clarfiying idententy between human and AI and improves security and data integrity which can help prevent server slow down and stolen sensitive inforamtion.

# Content
* Intro page teling what the quiz is about.
* (5 or 10)-question Quiz
* Random order of questions
* Different types of question in answering
* The quiz kicks out any user who answers wrong on thinking its the AI.
* Designs to help the application look less bland and dull.
* Ending page verifying the user is a human. 
# Goals
* Make sure application runs properly
* Test the quiz on an actual AI
* Test on Human
* have a good interface.
